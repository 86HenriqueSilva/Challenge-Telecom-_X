# Challenge-Telecom-_X
Projeto de Churn para Telecom X (Oracle ONE/Alura). Pipeline ETL, EDA e Deep Learning para prever evasão e ROI. Destaque para a recuperação de dados nulos e projeção financeira de 10 anos, provando um impacto estratégico de R$ 500k+. Foco em transformar dados brutos em decisões de negócio lucrativas.
