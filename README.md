 Telecom X: Estratégia de Retenção e Inteligência Preditiva
 Descrição do Projeto

Este projeto foi desenvolvido como desafio final do programa Oracle Next Education (ONE) em parceria com a Alura. O objetivo central é combater o Churn (Evasão de Clientes) da operadora Telecom X, utilizando técnicas avançadas de análise de dados, estatística e Machine Learning para proteger o faturamento da companhia.

O projeto não se limita a métricas técnicas; ele entrega um Plano de Ação Financeiro com projeções para os próximos 10 anos.
 Tecnologias e Ferramentas

    Linguagem: Python

    Bibliotecas de Dados: Pandas, NumPy

    Visualização: Seaborn, Matplotlib, OpenCV (Geração de Vídeo)

    Inteligência Artificial: Scikit-Learn (Redes Neurais MLP)

    Ambiente: Google Colab

 Estrutura do Projeto (Pipeline)

O desenvolvimento foi dividido em etapas modulares, seguindo as melhores práticas de Engenharia de Dados:
1. Engenharia de Dados e ETL

Extração de dados via API (JSON), normalização de dicionários aninhados e tradução completa para o português. Aplicação de princípios de Clean Code e monitoramento via logging.
2. O Resgate Estatístico dos "Vazios"

Utilização de análise de similaridade de perfil para provar que 224 clientes com status indefinido eram, na verdade, assinantes ativos.

    Resultado: Resgate imediato de R$ 14.204,35 em Receita Mensal Recorrente (MRR).

3. Diagnóstico de Risco (EDA)

Identificação do "Triângulo de Risco" da Telecom X:

    Fator Contratual: Clientes "Mês a mês" possuem 89% da taxa de evasão.

    Fator Tecnológico: A Fibra Óptica gera alta receita (R$ 292k), mas apresenta maior volatilidade.

    Fator Financeiro: O Cheque Eletrônico é o método com maior atrito.

4. Projeção de Impacto (Machine Learning)

Simulação de 10 anos utilizando Deep Learning para calcular o "Custo da Inércia".

    Cenário Sem Ação: Perda acumulada de R$ 3,77 milhões.

    Cenário Estratégico: Proteção de mais de R$ 500.000,00 através de ações de retenção.

 Resultados e ROI

A estratégia proposta foca no grupo de 1.628 clientes VIPs identificados como de alto risco.

    Eficiência Estimada: Redução da taxa de Churn de 25,7% para 15,8%.

    Impacto Financeiro: Economia acumulada de meio milhão de reais em uma década.
